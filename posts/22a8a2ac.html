一款 phpstorm markdown 主题

phpstorm 的 markdown 插件渲染的效果不符合我的需求故做了一下修改。使用方式是直接修改插件的默认样式。打开 Languages & Frameworks -> Markdown 找到
Add CSS rules 填入下面的 CSS 样式。
